# Portfolio Website

A responsive developer portfolio showcasing my projects, skills, and contact links.

## Live Demo
- Deployed on Vercel: <https://portfolio-website-eight-tau-30.vercel.app/>

## Features
- Responsive hero section with a clear personal tagline
- Projects section with links to source code
- About/Skills section
- Contact section (email + socials)
- Fast performance + SEO-friendly structure (depending on framework)

## Getting Started (Local Setup)

### 1) Clone the repository
```bash
git clone <REPO_URL>
cd <YOUR_REPO_FOLDER>
```

### 2) Install dependencies
```bash
npm install
```

### 3) Run the dev server
```bash
npm run dev
```

Open `http://localhost:3000` in your browser
